## По темам

### Стек и старт проекта

- Язык/типизация: TypeScript
  Фреймворк: React 18
  Сборка/Dev server: Vite
  Роутинг: react-router-dom
  Запросы к API / server-state: @tanstack/react-query (TanStack Query)
  Клиент для HTTP: нативный fetch + свой apiClient (Bearer accessToken + credentials: 'include' + авто-refresh через /auth/refresh)
  App-state: Zustand
  Формы: react-hook-form
  Валидация: zod + @hookform/resolvers/zod
  UI kit: MUI (@mui/material)
  Таблица: @mui/x-data-grid
  Тесты unit/integration: Vitest + @testing-library/react + @testing-library/user-event
  Моки API для тестов: MSW
  E2E (опционально, 1 happy-path): Playwright
  Линт/форматирование: ESLint + Prettier

- **Создай front-end проект** по этому стеку, **2 страницы**: логин и товары с таблицей.

### Авторизация, CORS, сессия, токены

- Ошибка при логине: `Failed to fetch`; CORS + `credentials: 'include'` + wildcard `Access-Control-Allow-Origin`.
- Чекбокс «Запомнить»: сессия после закрытия браузера vs сброс при закрытии вкладки — варианты с точки зрения безопасности.
- `credentials: 'include'` и localhost; **Vite dev proxy** — «да». (тут не работал логин с параметрами credentials: 'include', ругался на CORS, cursor предложил использовать Vite dev proxy чтобы это обойти)
- Повторы про remember-me / refresh; **expiresInMins 10** на логин и refresh; реализация refresh; хранение токенов; cookie `max-age` 10 мин при чекбоксе; вопросы про дублирование в `LoginPage`, `rememberMe` в типе `setSession`.
- Автозаполнение и фон поля; cross очищает форму, eye показывает пароль; (тут попросил добавить логику показа пароля при клике на "глаз")

### Архитектура кода и рефакторинг

- **Ошибки входа**: неверный логин или пароль (сообщения для пользователя).
- **Routes** в отдельной папке; переименовать **features → implementation**; **shared** распределить в **auth**.
- **Разбиение страниц** на компоненты — какая структура папок; нравится **первый вариант** (папка на страницу) — **сделать**.
- Логин: **всё про форму в одном файле**; страница с минимумом кода.
- Логин: **тонкая страница**, данные в **`useXxx` + React Query**, **UI без API**.
